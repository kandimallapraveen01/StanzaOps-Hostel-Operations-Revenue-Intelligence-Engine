| Column Name       | Business Meaning                                     |
| ----------------- | ---------------------------------------------------- |
| Resident_ID       | Unique identifier assigned to each resident.         |
| Resident_Name     | Full name of the hostel resident.                    |
| Gender            | Resident's gender.                                   |
| Date_of_Birth     | Resident's date of birth.                            |
| Mobile_Number     | Contact number of the resident.                      |
| Email             | Resident's email address.                            |
| Aadhaar_Number    | Government ID used for resident verification.        |
| Company_Name      | Company where the resident works (if applicable).    |
| Employee_ID       | Employee ID assigned by the company.                 |
| Occupation        | Resident's occupation or job role.                   |
| Check_In_Date     | Date the resident joined the hostel.                 |
| Check_Out_Date    | Date the resident left the hostel (if applicable).   |
| Room_Number       | Room allocated to the resident.                      |
| Bed_Number        | Bed assigned within the room.                        |
| Room_Type         | Type of room (Single, Double, Triple, etc.).         |
| Floor             | Floor where the room is located.                     |
| Monthly_Rent      | Monthly rent charged to the resident.                |
| Deposit_Amount    | Security deposit collected.                          |
| Amount_Paid       | Amount paid by the resident.                         |
| Balance_Amount    | Remaining amount to be paid.                         |
| Payment_Date      | Date of the latest payment.                          |
| Payment_Method    | Payment mode (Cash, UPI, PhonePe, Google Pay, etc.). |
| Payment_Status    | Current payment status (Paid, Partial, Pending).     |
| Occupancy_Status  | Indicates whether the bed is occupied or vacant.     |
| Resident_Status   | Resident's current status (Active, Vacated).         |
| Emergency_Contact | Emergency contact number.                            |
| Parent_Name       | Name of the parent/guardian.                         |
| Parent_Contact    | Parent/guardian contact number.                      |
| Permanent_Address | Resident's permanent address.                        |
| City              | Resident's city.                                     |
| State             | Resident's state.                                    |
| Pincode           | Postal code.                                         |
| Nationality       | Resident's nationality.                              |
| Religion          | Resident's religion (if collected).                  |
| ID_Proof_Type     | Type of identity proof submitted.                    |
| Aadhaar_Copy      | Indicates whether an Aadhaar copy is available.      |
| Voter_ID_Copy     | Indicates whether a voter ID copy is available.      |
| Joining_Date      | Date the resident officially joined.                 |
| Exit_Date         | Date the resident exited the hostel.                 |
| Bed_Status        | Status of the assigned bed (Occupied/Vacant).        |
| Room_Capacity     | Maximum number of residents allowed in the room.     |
| Available_Beds    | Number of beds currently available.                  |
| Remarks           | Additional notes related to the resident or room.    |
